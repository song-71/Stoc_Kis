#### 왜 이 시간대가 특별한가
```
- 상한가 직행 종목의 60~70%가 09:00~09:05에 상한가 도달
- 갭 상승 + 거래량 폭발 = 가장 강한 매수 시그널
- 단, 허수 급등 (09:01 상한가 → 09:03 급락)도 많음 → 필터 필수

## Part B. 종가매매 익일 폭락 방지 필터

---

### B-1. 익일 폭락 원인 분석

| 원인 | 빈도 | 특징 |
|------|------|------|
| **작전주 / 세력 이탈** | 높음 | 전일 거래량 급등 → 익일 시초가 갭 하락 |
| **재료 소멸** | 중간 | 뉴스/공시 1회성 → 다음날 관심 소멸 |
| **수급 고갈** | 중간 | 전일 상한가에서 매수 대기 물량 소진 |
| **외국인/기관 차익 실현** | 낮음 | 전일 매수 → 익일 시초가 매도 |

### B-2. 폭락 위험 필터 (종가매매 후보에 적용)

#### 15:18 종가매수 후보 선정 시 추가 필터

```
[기존 조건] (유지)
1. prdy_sign ∈ {1, 2} (상한가 또는 상승)
2. prdy_ctrt ≥ 20%
3. stck_prpr / stck_hgpr ≥ 0.97 (고가 대비 97% 이상)

[신규 추가 — 폭락 방지 필터]
4. 외국인 순매수 ≥ 0 (최근 3일 기준, frgnmem_pchs_trend)
   → 외국인 순매도 중이면 제외 (세력 이탈 가능성)

5. 체결강도 ≥ 120% (volume_power)
   → 체결강도 100% 미만이면 매도세 우세 = 위험

6. 거래대금 ≥ 50억원 (acml_tr_pbmn)
   → 거래대금 낮으면 유동성 부족 = 익일 매도 시 슬리피지

7. 상한가 잔량 비율 확인
   → 상한가에 대량 매수 잔량 = 매수세 지속 (익일에도 관심)
   → 상한가 잔량 적음 = 관심 소진 (위험)

8. 변동성 필터 (10일 vol)
   → vol > 5% (평소 널뛰기 종목) → 제외 또는 수량 축소
```

#### Signal Strength 적용 (종가매매용)

```
기본 점수: 0.50 (기존 4개 조건 통과)

[가산]
+0.15  외국인 3일 연속 순매수
+0.10  체결강도 상위 20%
+0.10  상한가 잔량 ≥ 전일 거래량의 20%
+0.10  저변동성 돌파 (10일 vol < 2%)
+0.05  기관 순매수

[감산]
-0.20  외국인 3일 연속 순매도 → 매수 금지 (0.5 미만 보장)
-0.15  체결강도 하위 30%
-0.10  상한가 잔량 < 전일 거래량의 5%
-0.10  고변동성 (10일 vol > 5%)

[결정]
0.6+   → 정상 수량 매수
0.5~0.6 → 최소 수량 매수
0.5 미만 → 매수 보류
```

### B-3. 시간외 단일가(16:00~18:00) 체결 종목 — 추가 검증

시간외 체결은 정규장보다 **유동성이 낮고 가격 왜곡 가능성** 높음.

```
[시간외 체결 종목 추가 조건]
1. 시간외 체결가 ≤ 정규장 종가 (갭 하락 없음)
   → 시간외에서 종가보다 비싸게 사면 이미 프리미엄 지불

2. 시간외 거래량 ≥ 정규장 거래량의 5%
   → 시간외 거래량이 극히 적으면 가격 신뢰도 낮음

3. 시간외 호가 스프레드 ≤ 2%
   → 스프레드 넓으면 익일 시초가 불확실
```

### B-4. 익일 리스크 관리 (종가매매/시간외 체결 종목)

```
